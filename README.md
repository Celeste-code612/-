食简溯时 · 古籍饮食数字档案
以数字之舟，渡千年食事之河

「食简溯时」是一个系统整理历代典籍中饮食记载的数字人文项目。我们收录宋、元、明三代古籍中的三百余道食方，以优雅的现代数字界面呈现，让千年滋味在指尖重现。

📖 项目概述
中华饮食文化源远流长，自先秦至明清，历代文人雅士留下大量珍贵的食单、食谱与饮食笔记。然而这些典籍或散落各处、或深藏高阁，寻常人难以亲近。

「食简溯时」是一扇通往古籍食事世界的数字之门。我们系统整理历代典籍中的饮食记载，借助 AI 智能体让千年食方在指尖重现，兼设社区供同好交流分享。一蔬一饭，皆是历史的滋味；一字一句，俱为文化的传承。

🗂️ 五大板块
板块	功能说明
芸编食简	古籍菜谱数据库。收录历代食典中的三百余道食方，可按朝代、书名、食材、菜名多维度检索浏览，每道菜附出处与原文。
味脉简谱	食脉流变图谱。以 VOSviewer 共现分析构建食材与菜品的关联网络，可视化呈现 657 个节点、4557 条共现关系。
馔迹简行	美食地理足迹。在地图上标记古籍食方的发源地与传播路径，按地域探索各地饮食风貌与名馔渊源。
试味简录	AI 古籍食谱智能体「食晓生」。输入食材或菜名，AI 从数据库中推荐匹配菜方，告之出处、朝代与烹饪之法。
吾味斋	用户个人中心。收藏食谱、发布打卡笔记、管理个人资料，与同好交流古今食事。
🛠️ 技术栈
前端框架：原生 HTML + CSS + JavaScript（纯静态页面）
字体：LXGW WenKai、Noto Serif SC、ChillLongCangKaiShu（自托管）
地图引擎：Leaflet（OpenStreetMap 底图）
数据可视化：Canvas 自绘 VOSviewer 风格知识图谱
AI 接口：DeepSeek API（对话式食谱推荐）
数据存储：localStorage（用户收藏与打卡数据）
